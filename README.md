# MagicalMechanics

## 🤝 貢献のルール (Contribution Guide)

本プロジェクトでは、ライセンス遵守と権利関係の明確化のため、すべてのコミットに **DCO (Developer Certificate of Origin)** 形式の署名を求めています。

### 署名のやり方
コミットメッセージの末尾に、以下の形式で署名を必ず含めてください。
`Signed-off-by: 名前 <メールアドレス>`

#### 1. Gitコマンドを使用する場合
コミット時に `-s` フラグを付けるだけで自動挿入されます。
```bash
git commit -s -m "コミットメッセージ"
```
#### プライベートなメールアドレスをさらしたくない場合
[https://github.com/settings/profile](https://github.com/settings/emails)からKeep my email addresses privateをOnに変更し表示されるアドレスを使用してください
